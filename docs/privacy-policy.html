Agent ScreenSaver — Privacy Policy

Last updated: March 15, 2026

Overview

Agent ScreenSaver is a status display app that connects to a local bridge server on your own network to show real-time AI coding agent activity. We respect your privacy and collect no personal data.

Data Collection

We do not collect, store, transmit, or share any personal data.

The app:

  • Connects only to a bridge server that you configure on your own local network or VPN
  • Does not send any data to our servers or any third-party services
  • Does not use analytics, crash reporting, advertising, or tracking SDKs
  • Does not access contacts, location, camera, microphone, or files
  • Stores only your bridge server URL locally on your device (in app preferences)

Network Communication

The app uses the INTERNET permission solely to connect to a bridge server at an address you provide. This connection is:

  • Initiated only by you (by entering a server URL and tapping Connect)
  • Directed only to an address you specify (typically a local network IP or VPN address)
  • Used to receive server-sent events (SSE) containing agent status information

No data is sent from the app to the bridge server. The connection is receive-only.

Subscriptions

Agent ScreenSaver offers optional paid subscriptions managed through Google Play. Payment processing is handled entirely by Google. We do not collect or store payment information.

Children's Privacy

This app is a developer tool not directed at children under 13. We do not knowingly collect information from children.
